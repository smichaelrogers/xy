### realtime sparklines

```shell
2.3.1 :113 > arr = []
2.3.1 :114 > n = 50
2.3.1 :115 > 20.times do
2.3.1 :116 >   n = n + 25 - rand(50)
2.3.1 :117?>   arr << n
2.3.1 :118?>   puts XY.chart :spark, arr
2.3.1 :119?>   sleep 0.2
2.3.1 :120?> end
```
